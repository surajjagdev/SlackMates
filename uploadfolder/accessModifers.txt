Access modifers allow different accesses to classes

Public, private, internal, protected

Public-- anywhere
private-- within same class
internal --access within same project(namespace).
virtual -- denote a method as overrideable. Can call base method and return a different value.

e.g. public virtual void UpdateName(string newName){

Name=newName;
}


in another class--
-----------------------------
public override void UpdateName(string Name){
base.UpdateName(newName);
Name="Some other value";
}
